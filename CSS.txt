display : inline
대표적인 태그로는 span을 예로 들 수 있음. 
text의 크기만큼만 공간을 점유하고 줄바꿈을 하지 않는다.

display : block
block 속성은 무조건 한줄을 점유하고 있고, 다음 태그는 무조건 줄바꿈이 적용된다. 
대표 태그로는 div가 있음.

display : inline-block
inline 속성의 특징과 block 속성의 특징 둘 다 가지고 있다. 
기본적인 특징은 inline 속성과 비슷한데(줄바꿈을 하지않고, 동일한 라인에 작성가능)
inline 속성에서 할 수 없었던 width/height 변경 및 line-height를 커스텀하게
적용할 수 있는 특징이 있다.
